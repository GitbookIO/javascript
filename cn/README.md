# Learn Javascript


这本书将教你Javascript编程的相关基础。不管你是菜鸟还是一个有经验的程序猿，这本书是为任何想学习Javascript编程的人而准备的。

JavaScript (*简称 JS *)是一种能够让网页响应用户交互的编程语言。它产生于1995年，现已是当下最火并且使用最广泛的语言之一。
